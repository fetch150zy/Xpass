## Purpose

你是否也有被各种账户密码支配的恐惧，太多的账号密码记不住怎么办？你可能需要一款密码管理软件来帮你做这件事。**Xpass**，使用**Flutter**实现（也许后续会增加基于**C++ Qt**的PC版本），一个支持多平台的密码管理软件，也许是你一直在寻找的。
